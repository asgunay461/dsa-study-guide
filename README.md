# DSA Study Guide

*Created on 2026-03-20T11:36:37Z*

## Overview
This repository provides a comprehensive guide for preparing for a Data Structures and Algorithms (DSA) final exam. It includes research notes, a detailed 4‑week study plan, and a dedicated branch for practice problem solutions.

## Structure
- `research-notes.md` – Summarized fundamentals and implementation guidance.
- `study-plan.md` – Week‑by‑week schedule.
- `practice-problems/` – Solutions to coding problems (on the `practice-problems` branch).
